Contact

Feedback, bug reports, dictionary issues, and policy concerns can be sent through the project issue tracker.

For missing words, please use the in-game suggestion flow first when possible. It records the exact word and language context for review. For broader issues such as broken layout, incorrect ranking behavior, privacy concerns, or advertising-placement concerns, open a GitHub issue.

Open a GitHub issue

What to include

  • The language and board size you were using.
  • The word or rule that looked wrong.
  • Your device and browser if the problem is visual or touch-related.
  • A screenshot when that helps explain the issue.
