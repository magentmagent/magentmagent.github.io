About

magentmagent games is a personal browser-game site for compact puzzles that can be played quickly on mobile or desktop.

The first game, word chain snake, combines word-chain rules with grid movement. Instead of typing words into a list, each word becomes part of a continuous path. The result is a word game that also asks the player to plan space, reuse matching crossings, avoid repeated words, and decide when to surrender for a final score.

The site is designed to grow beyond a single game. Shared pages such as this one, the game list, dictionary notes, privacy information, and contact information are kept separate from the live game screen so future games can be added with the same reviewable structure.

What makes the game original

The dictionaries come from external lexical resources, but the board rules, scoring, language-specific filters, snake mode, result sharing, ranking flow, and mobile interface are built for this site. Player suggestions are reviewed before becoming accepted words.

Current supported play

  • English, Korean, and Japanese game pages.
  • Separate rankings by board size and language.
  • Classic word-chain play and snake mode bonus play.
  • Manual review for suggested missing words.
